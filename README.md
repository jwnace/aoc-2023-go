# aoc-2023-go

Solutions for Advent of Code 2023 in Go

## Pre-requisites

- Install [Go](https://golang.org/doc/install) `1.21.4` or higher

### Mac OS

```shell
brew install go
```

### Windows

```shell
winget install GoLang.Go
```

### Linux

```shell
sudo apt install golang-go
```

## How to run the solutions

```shell
go run .
```

## How to run the tests

```shell
go test ./...
```
